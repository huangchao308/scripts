# Scripts

## Description

个人使用的脚本

## Usage

```bash
# build
make build

# 计算用户应该加的经验值
./bin/scripts cal

# 添加用户经验值
./bin/scripts add --add.action exp --config config # 如果是正式环境，则 --config config_prod

# 添加用户金币
./bin/scripts add --add.action coin --config config # 如果是正式环境，则 --config config_prod

```
